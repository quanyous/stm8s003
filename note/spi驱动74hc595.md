# spi驱动74hc595

## 有关spi：

1.  寻找相应的spi端口
2. io口开启相应的初始化（模式，电平，速度）
3. spi初始化（时钟，各种配置，使能）
4. 发送数据
5. 拉高nss片选电平，显示数据



## 74hc595

### 简介：8位的串行输入并行输出，实现IO口的扩展。

### 接口：

| stm8s003 | 74hc595 |
| :------: | :-----: |
|   SCK    |  SHCP   |
|   MOSI   |  DATA   |
|   NSS    |  STCP   |
